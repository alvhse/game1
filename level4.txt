Вводятся 10 натуральных чисел превышающих 2. Выведите произведение четных чисел и сумму нечетных. При отсутствии четных или нечетных чисел выведите 0.

i, m, k = 1, 0, 1
while i <= 10:
  x = input(f'Введите {i}-е число:')
  x = int(x)
  if x % 2 == 0:
    m += x
  else:
    k *= x
  i += 1
if k == 1:
  k = 0
print(f'Произведение - {k}, Сумма - {m}')
