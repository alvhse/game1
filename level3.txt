Вводятся 2 натуральных числа. Если первое число больше 10, то программа завершается и выводится «Слишком много». В противном случае выводится квадрат из символов, у которых число символов по ширине и высоте равно этому числу. Квадрат может состоять либо из «*», когда второе введённое число меньше 5, либо из «#» в противном случае.

a, b = input(''), input('')
a, b = int(a), int(b)
if a <= 10:
  i, s = 1, '*' if b < 5 else '#'
  while i <= a:
    print(s * a)
    i += 1
else:
  print('Слишком много')
