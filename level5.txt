Есть 2 переменные a и b, которые могут быть цифрами или символами (в программе не вводятся, просто существуют). Если обе переменных не являются цифрами, то выводится «Цифр нет». Если обе переменных являются цифрами, то выводится произведение цифр. Если только одна из переменных - цифра, то выводится квадрат из символов, где ширина/длина равны этой цифре, а сам квадрат состоит из символа-значения второй переменной.

if type(a) == str and type(b) == str:
  print('Цифр нет')
elif type(a) != str and type(b) != str:
  print(a * b)
elif type(a) != str:
  i, a = 1, int(a)
  while i <= a:
    print(a * b)
    i += 1
else:
  i, b = 1, int(b)
  while i <= b:
    print(a * b)
    i += 1
